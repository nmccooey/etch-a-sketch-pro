# etch-a-sketch-pro
Web app that allows a user to sketch a picture and adjust levels of shading and hue.

User is able to:
  - Add and remove grid lines
  - Adjust hue
  - Adjust saturation
  - Delete and create a new canvas with a specific grid size
  - Change pencil color
  - Undo/Redo changes

Made with Materialize CSS.
